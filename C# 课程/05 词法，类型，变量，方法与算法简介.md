# 标识符（Identifier）    
### 合法的标识符：
* 不能和关键字冲突  
* 可以数字，下划线，字母组成标识符，不能数字开头  
* 可用标识符，或者 @+关键字
* 标识符开始字符+后续字符（opt）
* 标识符开始字符可以数字，下划线，字母组成标识符，不能数字开头 
* 后续字符 可以是单个或者多个标识符（字符和数字，汉语等）
### 命名规范  
要是别人能读懂的定义  
* 类 是名词或者附属名词
* 属性 是名词或者附属名词
* 方法 是动词
### 大小写规范  
* 变量名 头单词首字符小写，后续字符首字母大写
* 类名，名称空间，方法  全部字符首字母大写

# 初识类型、变量和方法  
* 初识类型（type）  
  * 亦称数据类型（data type） 
  ```csharp
  static void Main(string[] args)
  {
        var x = 3L; //可以用var，也可以明确类型
        Console.WriteLine(x.GetType().Name);//Int64
        var y = 3;
        Console.WriteLine(y.GetType().Name);//Int32
  }
  ```
* 变量是存放数据的地方，简称“数据”  
  * 变量的声明  
  * 变量的使用  
  ```csharp
  static void Main(string[] args)
  {
      int x;//变量申明
      x = 100;//赋值，变量的使用 x=100L出错 
   }
  ```
* 方法（旧称函数）是处理数据的逻辑，又称“算法”  
  * 方法的声明  
  * 方法的调用  
```csharp
using System;

namespace ConsoleApp2
{
    class Program
    {
        static void Main(string[] args)
        {
            Calculator c = new Calculator();
            int x=c.Add(2, 3);//函数调用
            Console.WriteLine(x);
            string str = c.Today();
            Console.WriteLine(str);//19 
            c.PrintSum(4, 5);//9
        }
    }

    class Calculator
    {
        public int Add(int x,int y)//函数的定义
        {
            int result = x + y;
            return result;
        }
        public string Today()//没有数据输入，有输出
        {
            int day = DateTime.Now.Day;
            return day.ToString();
        }
        public void PrintSum(int x, int y)//有数据输入，没有输出
        {
            int result = x + y;
            Console.WriteLine(result);
        }
    }
}

```
* 程序=数据+算法  
  * 有了变量和方法就可以写有意义的程序了  

# 算法简介  
### 循环初体验  
```csharp
using System;

namespace ConsoleApp2
{
    class Program
    {
        static void Main(string[] args)
        {
            Calculator c = new Calculator();
            c.PrintXTo1(4);
        }
    }

    class Calculator
    {
       public void PrintXTo1(int x)
        {
            for(int i = x; i > 0; i--)
            {
                Console.WriteLine(i);
            }
        }
    }
}
```
### 递归初体验  
```csharp
using System;

namespace ConsoleApp2
{
    class Program
    {
        static void Main(string[] args)
        {
            Calculator c = new Calculator();
            c.PrintXTo1(4);
        }
    }

    class Calculator
    {
       public void PrintXTo1(int x)
        {
            if (x == 1)
            {
                Console.WriteLine(x);
            }
            else
            {
                Console.WriteLine(x);
                PrintXTo1(x - 1);
            }
        }
    }
}
```
### 计算1到100的和  
```csharp
class Calculator
    {
        public int SumFrom1ToX(int x)
        {
            int result = 0;
            for(int i = 1; i < x + 1; i++)
            {
                result += i;
            }
            return result;
        }
        public int SumFrom1ToX2(int x)
        {
            if (x == 1)
            {
                return 1;
            }
            else
            {
                int result = x + SumFrom1ToX2(x - 1);
                return result;
            }
        }
    }
```
